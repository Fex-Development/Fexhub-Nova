Web Hosting Documentation

Serve apps with managed NGINX, app stacks, and static hosting.

Managed NGINX

  • Auto SSL + HTTP/2; security headers enabled.
  • Upload your site to public/ or set an upstream for apps.

App Stacks (Node/Python/PHP)

Node

Set start command: node server.js or npm start. Use environment variables for secrets.

Python

Use a WSGI/ASGI server (gunicorn/uvicorn). Example: gunicorn app:app -b 0.0.0.0:3000.

PHP

Use PHP‑FPM with NGINX. Place code in document root and route via index.php.

Static Sites

Upload built assets to public/. Leverage edge caching. Configure redirects with an _redirects file if needed.
