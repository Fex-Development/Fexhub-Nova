About

Fexhub is a free, advanced HTML+CSS theme for hosting providers — built to be clean, fast, and drop-in simple.

Our Story

Started as Fexa in 2022, we built developer tooling, templates, and Pterodactyl/Dashactyl themes for hosting providers. That grew into a full site theme focused on speed, clarity, and modular content.

Timeline

  • 2022: Fexa project and Dashactyl themes.
  • 2023: Multi-egg for Pterodactyl (Discord, Python, Minecraft).
  • 2024: Hosting templates, bots, tools, OS in Python.
  • 2025: Fexhub theme releases with advanced styling and docs.

Values

Speed

No JS required for core UI. Pure HTML+CSS.

Clarity

Consistent components and typography.

Open

A theme designed to remix and extend.
