Domains, DNS & SSL

How do I point my domain to my server?

First, find your server’s public IP in the panel. In your domain registrar’s DNS settings, create an A record:

Type: A
Name/Host: @
Value: 203.0.113.10
TTL: 300 (or default)

For a subdomain (e.g., play.yourdomain.com) use:

Type: A
Name/Host: play
Value: 203.0.113.10
TTL: 300

IPv6? Add an AAAA record pointing to your IPv6 address. Propagation can take up to an hour.

How do I enable HTTPS (SSL) on my site or panel?

Point your domain first (A/AAAA). Then, in the panel, enable Auto-SSL. We issue a Let’s Encrypt certificate automatically after DNS resolves. If you use a CDN/proxy, make sure it allows HTTP (80) during validation.

What DNS records do I need for Minecraft?

Either point the root/subdomain with an A record to the server IP, or set up a SRV record if you don’t want to expose a port:

_minecraft._tcp.play.yourdomain.com  3600  IN  SRV  0 5 25565  play.yourdomain.com.
Do you support reverse DNS (rDNS) for VPS?

Yes. Open a ticket with your PTR hostname and the assigned IP. rDNS helps with mail deliverability and some API whitelists.

How long does DNS propagation take?

Usually a few minutes; up to 24 hours globally. Lower TTLs can speed up changes.

How can I migrate with zero downtime?

Create a snapshot → restore in the new region → test using a temporary subdomain → lower TTL → switch DNS → raise TTL after cutover.
